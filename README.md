SmartCart Clustering System is a machine learning project that uses unsupervised learning techniques to segment customers based on their shopping behavior. By applying clustering algorithms like K-Means and PCA, the system identifies meaningful customer groups to help businesses improve targeted marketing, personalized recommendations, and customer retention strategies.

This project performs data preprocessing, feature scaling, dimensionality reduction (PCA), and clustering to discover hidden patterns in customer purchase data. It also includes visualization of clusters and evaluation using metrics such as WCSS (Elbow Method).
